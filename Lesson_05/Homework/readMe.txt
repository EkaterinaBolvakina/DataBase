Lesson05: Homework:
Create a collection of cars, add about 10 cars to it, let some have the same year and some have the same brand. Example of one document:

  {
    brand: "BMW",
    price: 20,000,
    model: "X5",
    year: 2015,
    horsepower: 300,
  }
1. try crud operations: add, read, update, delete.
2. do one operation each for one and several elements
3. try displaying and sorting items through aggregator: display all items with price less than 20000, sort in reverse order.
4. add a related collection categories with types of cars: car, truck, pickup, convertible, etc.
5. output the values from the linked collections
6. show a random car.

------------------------------------- DE -----------------------------------------------
Erstellen Sie eine Sammlung von Autos, fügen Sie etwa 10 Autos hinzu, von denen einige das gleiche Baujahr und einige die gleiche Marke haben. Beispiel für ein Dokument:

  {
    Marke: "BMW",
    Preis: 20.000,
    Modell: "X5",
    Jahr: 2015,
    Pferdestärken: 300,
  }
1. CRUD Operationen ausprobieren: hinzufügen, lesen, aktualisieren, löschen.
2. je eine Operation für ein und mehrere Elemente durchführen
3. versuchen Sie die Anzeige und Sortierung der Elemente durch den Aggregator: zeigen Sie alle Elemente mit einem Preis von weniger als 20000 an, sortieren Sie in umgekehrter Reihenfolge.
4. Fügen Sie eine verknüpfte Sammlungskategorie mit Fahrzeugtypen hinzu: PKW, LKW, Pickup, Cabrio, etc.
5. Ausgabe der Werte aus den verknüpften Sammlungen
6. ein zufälliges Auto anzeigen.
